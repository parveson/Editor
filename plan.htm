Manuscript Review System

The purpose of this application is to support the review process for all files submitted for publication in a journal. The application technology is Microsoft ASP 3.0 using VBScript and ADO connections to an Access database on an IIS server.

The application has role-based access control, with strict separation between 3 types of users: editors, authors, and reviewers. Here are the functions supported:

Administrators/Editors control:

Editor data:

  • List all Editors
  • Add Editor
    • Store name and initial password in database
    • Notify by email
  • Delete an Editor

Author data:

  • List all Authors
  • Add Author
    • Store name and initial password in database
    • Notify by email
  • Delete an Author

Manuscript/file data:

  • List all files
    • Add editor's notes
  • Apply MR# to a file
  • Close file

Reviewer data:

  • List all Reviewers
  • Add Reviewer
    • Store name and initial password in database
    • Notify by email
  • Select and Assign a Reviewer to a MS
    • Select by disciplines, indicate workload
    • Authorize reviewer to download a MS and upload a Review
    • Add editor's notes
  • Delete a Reviewer

Review data:

  • List all reviews
  • Add review
    • Add editor's notes
  • Close Review

Disciplines data:

  • List all disciplines
  • Add discipline

Reminder data:

  • List reminders
  • Send reminder

Authors control:

  • Upload my MS or related files with notes
  • List my MSS and review status
  • Delete my file

Reviewers control:

  • List files I have been assigned to review
  • Upload my review with notes
  • List my reviews and status
  • Delete my file

All Users control:

  • Send my password
  • Change my password
  • View documentation
